PlaneApp

Simple web project built on Java Servlets & JSP.

The program has many useful features such as:
1) from the airport: 
      - add new plane
      - delete plane
      - delete all planes
      - read data from file.
2) from the user:
      - see the list of planes
      - search flights by: 
           -days (today, tomorrow, other day)
           -city (departure, arrival).
      - sorting planes by time of departure
